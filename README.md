A Ruby driver for Dexter Industries' BrickPi.

## Installation

Add this line to your application's Gemfile:

    gem 'brickpi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install brickpi

## Usage

    require 'brickpi'
